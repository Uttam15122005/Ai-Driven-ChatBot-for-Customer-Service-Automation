## 📌 About the Project

In today’s digital world, customers expect **instant and personalized support**.
This project introduces an **AI-powered chatbot** that automates customer service using **NLP + Machine Learning**, reducing workload and improving efficiency.

✨ Designed to handle real-time queries with intelligent responses and scalable architecture.

## ⚙️ System Workflow

```mermaid
graph TD
A[User Input] --> B[Greeting Check]
B -->|Yes| C[Send Greeting Response]
B -->|No| D[FAQ Matching]
D -->|Found| E[Return Answer]
D -->|Not Found| F[AI Response (TinyLlama)]
F --> G[Store Chat History]
G --> H[Display Response]
```

---

## 🧠 Algorithm

```bash
1. Receive user query
2. Preprocess input (lowercase, trim)
3. Check greeting
4. Search FAQ database
5. If not found → AI generates response
6. Save chat history
```
